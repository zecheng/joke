领导告诉我落评的十大理由

发表于 2015-09-07 分类于 校园
本文字数： 564 阅读时长 ≈ 1 分钟

font face=verdana>学校职称评定开始了，没个敢小视的。自以为工作出色、成绩过人的我，觉得没什么可担心的。最后，经过群众投票、领导评议，我居然落评了。经向领导讨教，得出我落评的十大理由如下： 一、我有自考本科文凭。领导说：“通过函授、自考现在不都拿到本科学历了吗？你并不突出。” 二、我的备课、作业等凡是业务工作，都合进度、满数量、有质量。领导说：“这是你应该干的，单凭它决定不了什么。” 三、我教着四个班的课，兼着班主任，工作量非同一般。领导说：“总体上大家基本差不多，你这方面多干了，别的方面肯定要少干。” 四、平日里我尊时守纪、兢兢业业、工作勤奋。领导说：“你就是刻板，学点灵活性行不？” 五、我虚心向学、服从领导、听从指挥。领导说：“唯命是从算不得什么优点，有创见才能合乎潮流。” 六、我关爱学生、热心辅导，深受学生的欢迎。领导说：“教学不能一味讨好，表面文章多了，反而华而不实。” 七、我与同事关系和谐友好，群众赞成度高。领导说：“工作还得看自己，群众关系仅是一部分，领导评议很重要。” 八、我的教学科研成果多，近两年就取得了9个证书。领导说：“科研成果对你很重要，但不能说就是对学校作出了突出贡献。” 九、我的教学成绩年年优秀。领导说：“学校领导哪个付出的不比你少？” 十、最后总结。领导说：“不是光干好工作就行的。”

我捡到了一只小鸟

发表于 2015-09-07 分类于 校园
本文字数： 515 阅读时长 ≈ 1 分钟

font face=verdana>
    天儿越来越热，下午的课上起来令人昏昏欲睡，昨天下午第一节课我听着听着就睡着了。课间被同桌摇醒，赶紧出去透透气、洗把脸精神精神。 
    洗洗脸的效果就是不一样，我一清醒，就发现水槽旁边有只小鸟儿，看样子好像是只燕子，大概是受了伤，翅膀上湿淋淋的，我把小鸟捧起来，怜惜地抚摸了两下。这时，上课铃响了，我想都没想捧着小鸟就跑回教室，把它藏在课桌里。 
    没过十分钟，瞌睡虫再次袭来，我使劲抵抗着、抵抗着……终于意志崩溃，趴在桌上睡着了。突然，我被一阵鸟叫惊醒，左看看右看看，发现鸟叫声是从自己课桌里传出来的！我偷偷地打开抽屉，一看，小鸟儿正在里面又叫又跳，我就赶紧把它从里面拿出来。我一摸它就不叫了，但眼睛依然很惊恐地盯着四周。 
    鸟叫声吸引了周围同学的注意力，两边的人纷纷要求我给他们看看，每个人见了小鸟儿都忍不住用手去摸它。小鸟儿显然不喜欢这样，别人一摸它就叫，于是教室里不时传出鸟叫声。越来越多的同学投来好奇的目光，老师也开始用眼瞪着我。 
    我赶紧把小鸟塞进抽屉，一心盼着赶快下课把它放了。这时不知怎么搞的，它又在课桌里叫了起来，我心虚地抬起头，看见老师正朝我走来，大声地说：“那位同学，请立刻把你的手机关掉！ ”
