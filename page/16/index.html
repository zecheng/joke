Linux忘记密码

发表于 2013-06-17 分类于 LINUX
本文字数： 523 阅读时长 ≈ 1 分钟

进入单用户模式，将/分区 mount为读写方式就可以改密码了。
CentOS系统中不同的运行级别（Run Level）代表了系统的不同运行状态，例如 Linux 服务器正常运行时处于运行级别3，是能够提供网络服务的多用户模式；而运行级别 1 只允许管理员通过服务器主机的单一控制台进行操作，即“单用户模式”。5是xwindows模式。

进入单用户模式的前提是CentOS系统引导器能正常工作。下面以 GRUB 为例，在 GRUB 启动菜单里有“a”、“e”和“c”三个操作按键，使用这三个按键均可进入单用户模式。
使用“a” 操作按键进入单用户模式：
在 GRUB 启动菜单里使用 “a” 操作按键编辑 kernel 那一行的参数，在行末输入一个空格，然后输入single ，以告诉 Linux 内核后续的启动过程需要进入单用户模式，回车即可进入 # 提示符的单用户模式。
root (hd1,0)
kernel /vmlinuz-2.6.18-194.8.1.el5 ro root=LABEL=/
initrd /initrd-2.6.18-194.8.1.el5.img

mount -o remount rw /

将/分区重mount为读写模式(否则为只读模式)
#passwd root
就可以更改密码了。
改完后重启即可。

vmware centos6.0 联网记

发表于 2013-06-17 分类于 LINUX
本文字数： 412 阅读时长 ≈ 1 分钟

我安装的是 centos6.0
一 首先查看在vmware 编辑-> 虚拟网络编辑器 ->vmnet8 中
vmnet信息(需要选择上/一般默认都是选着上的)

  1. NAT
  2. 连接主机虚拟适配器名称:Vmware网络配置器vmnet8(这个是默认的也可以自己设定)
  3. 使用本地DHCP服务来分配虚拟机的IP地址
    如果不正确就把相应的选上
    二 在vm窗口右下方有个网络连接(两个小电脑)双击选择NAT确定

三 在centos6.0中 输入
vim /etc/sysconfig/network-scripts/ifcfg-eth0 回车

查看文件如果没有我写的相应命令自己添加上

DEVICE=”eth0” #网卡
HWADDR=”这里面是你的MAX地址”
NM_CONTROLLED=”yes”
ONBOOT=”yes”
BOOTPROTO=dhcp

保存上

service network restart 从启网络

ping www.ioqx.com 看看是不是有数据了 哈哈O(∩_∩)O~
